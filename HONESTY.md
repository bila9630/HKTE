# HONESTY.md

## Development Declaration

This project (HKTE - Hong Kong Truck Explorer) was developed entirely during the hackathon timeframe.

### Prior Experience

We have prior experience working with Mapbox GL from other projects. This experience includes:

- Previous implementations of Mapbox GL maps in different contexts
- Familiarity with Mapbox's API, styling system, and 3D features
- Understanding of map projections, camera controls, and layer management
- Experience with GeoJSON data structures and map animations

### Hackathon Development

Despite our prior Mapbox experience, **all code for this project was written during the hackathon with the help of coding assistants**. This includes:

- Every line of React/TypeScript code
- All Mapbox GL integration and custom logic
- Truck animation algorithms and route calculations
- UI components and styling
- Project configuration and setup

No pre-existing Mapbox code or templates were copied or reused from previous projects. While our experience helped us work faster and make informed technical decisions, the implementation is original to this hackathon submission.

### Technical Choices

Our familiarity with Mapbox influenced our decision to use it for this project, as we knew it could deliver the 3D visualization and animation features we envisioned. However, the specific implementation, route logic, and integration with TanStack Start were all developed during the hackathon.
